# Generate-a-QR-code-in-Node.js

This code will enable you to generate a QR code in Node.js from URLs and Text and preview it as a simple webpage.

## Steps

- Initialize project
- Configure nodemon in package.json
- Add a starting point
- Add a views folder
- Run the project
